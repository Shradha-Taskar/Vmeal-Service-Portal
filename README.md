# Vmeal-Service-Portal
VMeal Service Portal demonstrates backend development skills using Java Spring Boot with database integration for a real-world food service application.
VMeal Service Portal is a Meal Subscription and Tiffin Service Management Application designed to manage customers, meals, and subscriptions efficiently. The system helps streamline daily tiffin services by maintaining customer details, meal plans, and subscription records in a structured way.
This project is built with a backend-focused architecture and reflects a real-world food service business workflow.
